# Meditation App

This is a meditation app that allows users to start guided meditation sessions and read positive affirmations. It is built using React Native and Expo.

## Get started

1. Install dependencies

   ```bash
   npm install
   ```

2. Start the app

   ```bash
    npx expo start
   ```
